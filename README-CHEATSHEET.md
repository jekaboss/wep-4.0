# Шпаргалка - Сервер для збереження

## Як запустити

### 1. Встановіть Node.js
Якщо не встановлено, завантажте з: https://nodejs.org/

### 2. Встановіть залежності
Відкрийте командний рядок у папці проекту і виконайте:
```bash
npm install
```

### 3. Запустіть сервер

#### Варіант A: Одноразовий запуск
Двічі клацніть на **`start-server.bat`**

Або через командний рядок:
```bash
npm start
```

#### Варіант B: Автоматичний запуск при завантаженні Windows
1. Двічі клацніть на **`add-to-startup.bat`**
2. Тепер сервер запускатиметься автоматично при кожному старті Windows

#### Варіант C: Видалити з автозапуску
Двічі клацніть на **`remove-from-startup.bat`**

### 4. Відкрийте сторінку

#### Варіант A: Автоматичний запуск (рекомендовано)
Відкрийте **`shpargalka.html`** у браузері
- Сторінка сама перевірить чи сервер працює
- Якщо ні - запропонує запустити
- Після запуску автоматично відкриє шпаргалку

#### Варіант B: Прямий доступ
Якщо сервер вже запущено, відкрийте **`cheatsheet.html`**

## Як це працює

1. Сервер створює папку `cheatsheets-data/` у проекті
2. При додаванні/редагуванні/видаленні шпаргалки, файл `cheatsheets.json` автоматично оновлюється
3. Всі дані зберігаються у `cheatsheets-data/cheatsheets.json`

## Структура папок

```
wep-2.0-main/
├── cheatsheet.html      # Сторінка шпаргалки
├── cheatsheet.js        # JavaScript
├── server.js            # Node.js сервер
├── package.json         # Залежності
├── cheatsheets-data/    # Папка з даними (створюється автоматично)
│   └── cheatsheets.json # Ваші шпаргалки
└── README-CHEATSHEET.md # Цей файл
```

## API Ендпоінти

- `GET /api/cheatsheets` - Отримати всі шпаргалки
- `POST /api/cheatsheets` - Додати шпаргалку
- `PUT /api/cheatsheets/:id` - Оновити шпаргалку
- `DELETE /api/cheatsheets/:id` - Видалити шпаргалку
- `POST /api/cheatsheets/save` - Зберегти всі шпаргалки

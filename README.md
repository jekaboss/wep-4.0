**Опції:**
- 1 - Запустити сервер (фоновий режим)

- 3 - Запустити Electron додаток
- 4 - Запустити з мережевим доступом

- 7 - Відкрити у браузері

## 🔧 Вирішення проблем

### ❌ "Node.js не встановлено"

**Рішення:** Встановіть Node.js з https://nodejs.org/

### ❌ "npm install" не працює

**Рішення:**
```bash
# Очистіть кеш npm
npm cache clean --force

# Видаліть node_modules і спробуйте знову
rm -rf node_modules package-lock.json
npm install
```

### ❌ Сервер не запускається

**Рішення:**
1. Перевірте чи порт 3000 не зайнятий
2. Запустіть від імені адміністратора
3. Перевірте антивірус/фаєрвол

### ❌ Не можу підключитися з телефону

**Рішення:**
1. Переконайтеся що пристрій в тій самій Wi-Fi мережі
2. Дозвольте порт 3000 у фаєрволі
3. Перевірте IP-адресу ПК

### ❌ Electron не запускається

**Рішення:**
```bash
# Перевстановіть залежності
npm install
npm install electron --save-dev
```
